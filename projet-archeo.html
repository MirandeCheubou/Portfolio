Développement

Fonctionnalités clés

Authentification sécurisée

Inscription avec génération de mot de passe via script Python, hachage password_hash(), vérification par password_verify() et gestion de session.

Base de données MySQL

3 tables reliées : formulaire, inscription, actualités. Accès via PDO avec requêtes sécurisées.

Actualités dynamiques

Les articles sont stockés en BDD. Un utilisateur non connecté voit 3 articles. L'accès complet est réservé aux membres inscrits.

Panel administration

Interface admin avec 3 tableaux de données (contacts, inscriptions, actualités). Suppression des entrées en un clic via formulaires POST.

Mode jour / nuit

Toggle CSS via classe .night sur le body. Préférence mémorisée en localStorage entre les pages.

Design responsive

Mise en page adaptée mobile/tablette/desktop via CSS Grid, Flexbox et media queries. Thème archéologique cohérent avec police Cinzel et palette parchemin.
